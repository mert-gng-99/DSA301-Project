Gestalt & Bertin Principles — Applied in All 3 Sketches

Gestalt Principles
  • Proximity: Related metrics are grouped together (sleep stages, HR zones, metric cards)
  • Similarity: Same ring shape across all 3 sketches signals "score" data type
  • Continuity: Sparklines and trend bars show temporal flow naturally
  • Figure-Ground: Dark background isolates bright metric cards as foreground figures
  • Closure: Rounded cards create complete perceptual units per data group
Bertin's Retinal Variables
  • Size: Ring radius encodes score magnitude (primary variable)
  • Hue: Green = good recovery, Red = low, Blue = strain (semantic mapping)
  • Saturation: Calendar heatmap uses saturation intensity for scores
  • Length: HR zone bars and sparklines encode quantitative values
  • Position: Y axis and bar height always used for the main quantity
